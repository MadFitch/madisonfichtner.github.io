Raspberry Pi Drone and Google Home

Raspberry Pi is an incredible platform for developers, and creators alike to explore technology. I decided to try out a few projects using first, open source software to power a quadcopter, and second, Googles API for their Google Home.

The quadcopter used the combination of Python programs and servo motors controlled with an Arduino microcontroller. The drone was controlled through a wireless connection with the use of an android application.

Google recently came out with an openly downloable API for their Google Homes. The only other necessary component for creating your own Google Pi is connecting the Pi to Google's servers
